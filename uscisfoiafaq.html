USCIS FOIA Request FAQ

Log in to your USCIS account Log in to “FIRST” Select “Request for Myself” Enter Country of Birth and Date of Birth Enter Alien # (available on any visa documents, LPR card, notices from agencies) Enter I-94 Arrival-Departure Record Enter Passport Number Select the document you are requesting from the “Available File Types” dropdown menu Select “Alien File” if you wish to request a copy of your entire file Select yes to “Do you have upcoming Immigration Court proceedings” if you do Digitally sign the form with your name and the date
